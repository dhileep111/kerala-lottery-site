# Kerala Ticket Results — GitHub Pages

  A fully static Kerala Lottery result website, SEO-optimised for Google Search Console.

  ## Files
  - `index.html` — Homepage with today's result, schedule, quick links
  - `karunya.html`, `karunya-plus.html`, etc. — Individual lottery result pages
  - `guessing-numbers.html` — Lucky numbers in Tamil
  - `claim-guide.html` — How to claim lottery prize
  - `about.html` — About page with full disclaimer
  - `sitemap.xml` — XML sitemap for Google Search Console
  - `robots.txt` — Search engine crawl rules
  - `assets/style.css` — Shared stylesheet

  ## Setup for GitHub Pages
  1. Upload all files to a GitHub repository
  2. Go to Settings → Pages → Source: `main` branch, `/ (root)`
  3. Your site will be at `https://YOUR-USERNAME.github.io/REPO-NAME/`
  4. Replace `YOUR-USERNAME` and `kerala-ticket-results` in all canonical URLs and sitemap
  5. Submit `sitemap.xml` to Google Search Console

  ## SEO Features
  - Unique `<title>` and `<meta description>` on every page
  - Canonical URLs on every page
  - Open Graph tags
  - Schema.org JSON-LD structured data (Article, HowTo, WebPage)
  - Breadcrumb navigation (HTML + Schema.org)
  - `sitemap.xml` ready for Google Search Console submission
  - `robots.txt` with sitemap reference
  - Semantic HTML (`<main>`, `<nav>`, `<article>`, `<footer>`, `<section>`)
  - No JavaScript required for content — Googlebot reads all text directly
  - Tamil language content with `font-family: Noto Sans Tamil`
  - Mobile-first responsive design
  - Dark mode support
